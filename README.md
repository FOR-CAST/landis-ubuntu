# `landis-ubuntu`

LANDIS-II model (v8) and extensions modified to run on Ubuntu 24.04 LTS

## Prerequisites

1. Install v8.0 of the .NET framework:

    sudo apt-get install -y \
      dotnet-sdk-8.0 \
      git \
      libjpeg62 \
      libpng16-16t64 \
      software-properties-common \
      unzip \
      wget
